# Daniels

A responsive personal portfolio website built using **HTML**, **CSS**, and **Bootstrap**.

## 📌 Description
Daniels is a clean and modern portfolio template perfect for showcasing your skills, projects, and personal information.  
It includes sections like About, Services, Portfolio, Testimonials, and Contact.

## 🚀 Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5

## 💡 Features
- Fully responsive design  
- Smooth scrolling navigation  
- Modern and clean UI  
- Portfolio gallery section  
- Contact form UI (front-end)
